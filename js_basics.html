Scope

Scope defines where variables and functions are accessible in your code. There are two types: Global Scope and Local Scope.

Closures

A closure is a function that has access to its outer functionâ€™s scope even after the outer function has returned.

Promises

Promises are objects that represent the eventual completion or failure of an asynchronous operation. They allow handling asynchronous code more efficiently.

Async/Await

Async functions allow the use of await to pause execution until a promise is resolved or rejected, simplifying asynchronous code.

Hoisting

In JavaScript, variable and function declarations are moved to the top of their scope before code execution. This is called hoisting.
